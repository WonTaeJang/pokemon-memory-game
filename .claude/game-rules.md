# 포켓몬 카드 뒤집기 게임 규칙

## 게임 개요
카드를 뒤집어 같은 포켓몬 쌍을 찾는 메모리 매칭 게임.
모든 스텝을 순서대로 클리어하면 게임 완료.

---

## 스텝 구성

| 스텝 | 그리드 | 카드 수 | 쌍 수 | 허용 뒤집기 횟수 |
|------|--------|---------|-------|----------------|
| 1    | 2×2    | 4장     | 2쌍   | 10회           |
| 2    | 3×4    | 12장    | 6쌍   | 10회           |
| 3    | 4×5    | 20장    | 10쌍  | 10회           |
| 4    | 6×6    | 36장    | 18쌍  | 10회           |

---

## 기본 규칙

- 모든 카드는 뒷면으로 시작
- 한 번에 카드 2장을 뒤집을 수 있음
- 2장이 같은 포켓몬이면 매칭 성공 → 카드 앞면 유지, 뒤집기 카운트 소모 없음
- 2장이 다른 포켓몬이면 매칭 실패 → 1초 후 자동으로 뒤집힘, 뒤집기 카운트 1 소모
- 허용 뒤집기 횟수를 모두 소모하면 게임 오버

---

## 진행 흐름

1. 스텝 시작 → 카드 셔플 후 뒷면으로 배치
2. 플레이어가 카드 2장 선택
3. 매칭 성공 → 해당 쌍 앞면 유지, 계속 진행
4. 매칭 실패 → 1초 후 카드 뒤집힘, 뒤집기 카운트 -1
5. 모든 쌍 매칭 완료 → 스텝 클리어, 다음 스텝으로 이동
6. 뒤집기 카운트 0 → 게임 오버

---

## 점수

- 점수 = 스텝 클리어 시 **남은 뒤집기 횟수** 합산
- 4스텝 모두 클리어 시 최종 점수 = 각 스텝 남은 횟수의 합 (최대 40점)

---

## 포켓몬 이미지

- PokeAPI를 통해 포켓몬 이미지를 가져옴
- 전체 게임에서 사용되는 포켓몬 36마리는 중복 없이 랜덤 선정
- 스텝 간 포켓몬 중복 없음
- 포켓몬 배정: 스텝1(2마리), 스텝2(6마리), 스텝3(10마리), 스텝4(18마리)

---

## 게임 클리어 / 게임 오버 화면

**클리어 시**
- 최종 점수 표시 (총 남은 뒤집기 횟수 합산)
- 각 스텝별 남은 횟수 표시
- 재시작 버튼

**게임 오버 시**
- 실패한 스텝 표시
- 획득한 점수 표시
- 재시작 버튼
